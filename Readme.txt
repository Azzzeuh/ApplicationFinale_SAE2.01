Pour compiler :
Placez-vous à la racine du répertoire ApplicationFinale_SAE2.1
Lancez la commande : javac @compile.list -d class

Pour exécuter :
Placez-vous dans le répertoire class
Lancez la commande : java src.Controleur


pour exécuter un scénario on effectue 

Java src.scénarioX

X est le numéro de scénario
